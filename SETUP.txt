===============================================================

  FIX: Install Python from python.org
       Make sure to check "Add Python to PATH" during install

ERROR: No module named 'torch'
  FIX: Make sure (venv) is showing, then run:
       pip install -r requirements.txt

ERROR: pip is not recognized
  FIX: Try:  py -m pip install -r requirements.txt

ERROR: Out of memory
  FIX: Open app.py, change test_size=200 to test_size=50

Everything runs slowly
  NORMAL: Without a GPU, CPU inference is slower.
  The benchmark will still complete and results will be correct.
